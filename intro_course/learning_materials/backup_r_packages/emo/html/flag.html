Flag emoji

Description

Flag emoji

Usage

flag(pattern)

Arguments

pattern

pattern suitable for [stringr::str_detect] matched against the name of the flag

Value

If the pattern matches a single flag, the emoji ( with classes "flag" and "emoji" ) is returned

Examples

## Not run: 

flag( "^Fra" )
flag( "New Zealand" )

# name of all the flags
if( require(dplyr) ){
 emo::jis %>%
   filter( group == "Flags" ) %>%
   pull(name)
}
